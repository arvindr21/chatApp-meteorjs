ChatApp powered by Meteorjs
===================

A simple chat built using Meteorjs.

Demo : http://jojschatapp.meteor.com

Setup
------

1. Download and Install Meteorjs
```bash
$ curl https://install.meteor.com/ | sh
```
2. Clone/Downlaod the repo
3. To run the app
```bash
$  meteor
```
